# 香港醫管局醫療費用計算器

一個幫助你預計全年醫療費用的簡易工具。

---

## 這個工具可以幫你什麼？

- 💰 **預計全年醫療費用** – 知道今年要花多少錢去睇醫生
- 📅 **看清楚費用分佈** – 知道哪幾個月費用會特別高
- 🎯 **及早了解保障上限** – 提醒你醫管局有 HK$10,000 全年上限，但需要主動申請

---

## 誰適合用？

✓ 需要同時睇多個專科的病人  
✓ 長期服藥需要補藥的人  
✓ 想了解今年醫療費用的家人  
✓ 醫務社會服務人員  

---

## 怎樣開始使用？

只需要告訴我們：

1. **你睇緊的專科** 和 **下次覆診月份**（你的HA go或覆診紙上會寫）
2. **覆診頻率** – 多久睇一次（每月、每2個月等等）
3. **有無長期用藥** – 如果有，大概用多少種

所有收費金額我們已經內建了，你不需要自己輸入。

---

## 重要提醒

### 關於全年收費上限（HK$10,000）

🔴 **最重要的一點：** 這個上限**不是自動的**

- 需要你**主動向醫管局申請**
- 需要經過**審核才會生效**
- 上限是以**曆年計算**（1月1日至12月31日）

這個工具會提醒你可能接近上限，但實際資格及申請程序需要向醫管局或醫務社會服務部查詢。

---

## 重要聲明

- 本計算器**僅供估算參考**
- 實際收費以醫管局系統為準
- 不作為資格判定或醫療建議
